# Екологічна безпека атмосферного повітря: моніторинг PM₁₀ у Кривому Розі

**Дисертація на здобуття наукового ступеня кандидата технічних наук**  
Спеціальність: Екологічна безпека  
Місто: Кривий Ріг, 2025–2026

---

## Про дослідження

Мета дослідження — визначити частку фракції PM₁₀ у загальному завислому пилу (TSP)
в умовах промислового міста Кривий Ріг та побудувати програмно-алгоритмічну модель
корекції індикативних вимірювань PM₁₀ з урахуванням метеорологічних чинників
(метод Random Forest).

### Джерела даних
| Система | Параметр | Пункти | Період |
|---|---|---|---|
| Cairnet ENVEA (оптичні сенсори) | PM₁₀, PM₂.₅, T, RH, WS | pan2, pan5, pan6 | 2025 р. |
| ТЗА — гравіметрія (РД 52.04.186-89) | TSP (мг/м³) | TZA_Post3, TZA_Post7 | 2025 р. |

### Ключові результати (Розділ 4)
- Синхронізована вибірка: **518 пар** TSP–PM₁₀ (TZA_Post3 + pan2)
- Допоміжна вибірка: **465 пар** (TZA_Post7 + pan5)
- Повнота ряду pan2: **82.9%** (технічні проблеми жовтень–листопад)
- Бутстреп-аналіз: 95% ДІ річного середнього PM₁₀ = **(25.4; 29.4) мкг/м³**

## Як запустити скрипти

### 1. Встановити Python 3.10+
Завантажити з [python.org](https://www.python.org) → при встановленні поставити галочку **"Add to PATH"**

### 2. Встановити залежності
```bash
pip install -r requirements.txt
```

### 3. Запустити скрипти (з кореневої папки репозиторію)
```bash
# Бутстреп-аналіз репрезентативності вибірки
python run_bootstrap.py

# Аналіз викидів методом Тюкі (відтворює Таблицю 4.13)
python run_tukey.py

# Тест Колмогорова–Смирнова
python run_ks_test.py
```

> **Примітка:** скрипти очікують файл `Додаток_4_3.xlsx` у тій самій папці.

## Умовні позначення

| Позначення | Опис |
|---|---|
| pan2, pan5, pan6 | Сенсори Cairnet (пункти № 3, 5, 6) |
| TZA_Post3 | Пост ТЗА № 3 (основний, просторово суміщений з pan2) |
| TZA_Post7 | Пост ТЗА № 7 (допоміжний, суміщений з pan5) |
| TSP | Total Suspended Particles — загальний завислий пил (гравіметрія) |
| PM₁₀ | Зважені частинки ≤ 10 мкм (оптична нефелометрія) |
| RF | Random Forest — метод машинного навчання |
